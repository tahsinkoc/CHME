# CHME Overview

CHME is a compact hierarchical memory engine designed for deterministic retrieval.
It organizes information into document roots, sections, and chunks.

## Core Principles

- Compact architecture
- In-memory indexing
- Deterministic behavior
- Dependency-light implementation

### Retrieval Strategy

The current strategy is keyword-based retrieval with section-aware ranking.
Future semantic ranking can be added without changing core storage.
