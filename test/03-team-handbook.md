# Team Handbook

## Engineering Expectations

Code should remain simple and readable.
Avoid over-engineering and unnecessary abstractions.

## Testing

Test scripts should be easy to run and should verify ingest and retrieval behavior.
When possible, tests should use deterministic inputs and deterministic scoring.

### Operations

For local inference, teams can use a local 7B model endpoint.
For hosted inference, OpenAI-compatible APIs can be used.
